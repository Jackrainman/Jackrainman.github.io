# About

这里不是完整履历页，而是这个文档站的使用说明和内容定位。

## 这个站点在写什么

这个仓库主要记录我在机器人电控方向的长期整理，重点放在这些主题：

- 控制理论与算法选型
- 底盘运动学与速度规划
- CAN 通信与电机驱动接口
- STM32 / FreeRTOS 工程实践

相比“知识点摘抄”，这里更强调两件事：

1. 先把问题分层，再决定该看哪类算法或方案。
2. 尽量把抽象概念落到可实现、可调试、可复用的工程表达上。

## 首页为什么只展示一部分

`docs/zh` 同时作为 Obsidian Vault 和 MkDocs 内容目录使用。

适合公开发布的专题文档放在普通专题目录中；私人、草稿和归档资料放入 `_Private/`、`_Drafts/`、`_Archive/`，这些目录不进入 Git，也不参与网站构建、导航和搜索。

## 建议怎么读

- 第一次进入：先看 [控制理论学习路径指南](../ControlTheory/00-学习路径指南.md)
- 想看底盘与运动控制：读 [底盘解算](../Chassis/底盘解算.md) 和 [速度规划-理论](../SpeedPlan/速度规划-理论.md)
- 想看通信与工程实现：读 [CAN 总线基础](../CAN/CAN总线基础.md) 和 [电机驱动代码详细讲解](../Motor/电机驱动代码详细讲解.md)

## 内容边界

- 首页推荐区只放公开价值高、通用性强的主题
- 培养任务、考核标准、Legacy 和废弃文档不进入公开站点
- 私人笔记和内部材料保留在 `docs/zh` 的下划线目录中，只作为本地 Obsidian 资料使用
