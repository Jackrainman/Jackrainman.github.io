# About

这里不是完整履历页，而是这个文档站的使用说明和内容定位。

## 这个站点在写什么

这个仓库主要记录我在机器人电控方向的长期整理，重点放在这些主题：

- 控制理论与算法选型
- 底盘运动学与速度规划
- CAN 通信与电机驱动接口
- STM32 / FreeRTOS 工程实践

相比“知识点摘抄”，这里更强调两件事：

1. 先把问题分层，再决定该看哪类算法或方案。
2. 尽量把抽象概念落到可实现、可调试、可复用的工程表达上。

## 首页为什么只展示一部分

`docs/zh` 目录里除了专题文档，也保留了培养方案、阶段任务、文档规范、历史归档和一些内部使用材料。

这些内容仍然保留在站内，但首页只展示更适合公开阅读、也更能代表本站质量的精选内容，避免第一次进入时被完整资料树淹没。

## 建议怎么读

- 第一次进入：先看 [控制理论学习路径指南](../ControlTheory/00-学习路径指南.md)
- 想看底盘与运动控制：读 [底盘解算](../Chassis/底盘解算.md) 和 [速度规划-理论](../SpeedPlan/速度规划-理论.md)
- 想看通信与工程实现：读 [CAN 总线基础](../CAN/CAN总线基础.md) 和 [电机驱动代码详细讲解](../Motor/电机驱动代码详细讲解.md)

## 内容边界

- 首页推荐区只放公开价值高、通用性强的主题
- 培养任务、考核标准、Legacy 和废弃文档不作为第一页展示重点
- 完整资料仍然通过导航和搜索保留访问路径
