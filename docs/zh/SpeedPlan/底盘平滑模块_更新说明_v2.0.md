# 底盘平滑模块更新说明（v2.0）

## 版本信息

- 模块：`chassis_calculations`
- 旧版本：`v1.0`（2025-11-22）
- 新版本：`v2.0`（2026-03-02）
- 目标：在保留 1.0 调用方式的前提下，提升底盘速度平滑性并降低机械过冲风险

---

## 1. 与 v1.0 的核心差异

### 1.1 控制周期计算修复（关键）

- v1.0 问题：`dt` 直接使用 tick 差值，未换算为秒，导致加速度计算量纲错误。
- v2.0 更新：统一使用 `dt = delta_ticks / tick_freq_hz`，并加入异常保护（`dt<=0` 或 `dt>0.1s` 直接保持输出）。

### 1.2 S 曲线从占位实现升级为可用实现（关键）

- v1.0 问题：S 曲线为 TODO/占位逻辑，不具备有效 jerk 约束。
- v2.0 更新：
  - 引入 jerk 限制的 S 曲线单轴更新；
  - 增加“减速前瞻”逻辑，在接近目标时提前回收加速度，降低速度过冲。

### 1.3 余弦平滑升级为完整时间域过渡

- v1.0：近似余弦平滑，目标变化时行为不够一致。
- v2.0：每轴独立维护余弦过渡状态，按完整时间参数推进，速度起止更平滑。

### 1.4 增加速度硬限幅

- v1.0：没有统一三轴速度上限约束。
- v2.0：新增 `x/y/w` 速度上限钳位，避免异常指令导致速度超界。

### 1.5 防侧翻限速保留并参数化

- v1.0：始终启用固定常量。
- v2.0：保持算法一致，但可配置开关和参数（`base/max/threshold`）。

### 1.6 兼容性改进

- 保留了 v1.0 的主接口：
  - `chassis_speed_plan_init`
  - `chassis_speed_plan`
  - `chassis_get_speed_plan`
  - `easy_speed_calc_handle/cosine_speed_calc_handle/scurve_speed_plan_handle`
  - `speed_point_limit`

---

## 2. 新增可选配置接口

在不影响旧代码调用的情况下，新增以下配置 API：

- `chassis_speed_plan_set_tick_freq(uint32_t tick_freq_hz)`
- `chassis_speed_plan_set_accel(float accel_x, float accel_y, float accel_w)`
- `chassis_speed_plan_set_jerk(float jerk_x, float jerk_y, float jerk_w)`
- `chassis_speed_plan_set_speed_limit(float speed_x, float speed_y, float speed_w)`
- `chassis_speed_plan_set_rollover(int enable, float base_limit, float max_limit, float threshold)`

说明：
- `jerk <= 0` 时自动按 `10 * accel` 推算；
- `speed_limit <= 0` 表示该轴不限制。

---

## 3. 默认参数（v2.0）

- tick 频率：`1000 Hz`
- 最大加速度：`x=40, y=40, w=20`
- 最大 jerk：自动推算（默认 `10 * accel`）
- 速度上限：`x=5000, y=5000, w=10`
- 防侧翻：默认开启，`threshold=30, base=30, max=40`

---

## 4. 迁移建议（从 v1.0 到 v2.0）

1. 直接替换 `chassis_calculations.h/.c`，旧调用可保持不变。  
2. 初始化后建议显式设置控制周期：
   - `chassis_speed_plan_set_tick_freq(configTICK_RATE_HZ);`
3. 优先使用 `CHASSIS_SPEED_PLAN_S_CURVE` 作为默认模式，减少机械冲击和过冲。  
4. 根据底盘实测调整：
   - 加速度先小后大；
   - jerk 先默认自动，再按振动情况细调；
   - 防侧翻阈值根据底盘重心和轮距调参。

---

## 5. 风险与注意事项

- 本次升级改变了 `dt` 计算方式，实际加减速会比旧版“更物理一致”，调参需重新标定一次。  
- 如果上层任务周期抖动较大，建议先确保控制任务频率稳定，再调 jerk 和 rollover 参数。  
- 当目标速度变化极快时，S 曲线会优先保证平滑性，不会像 v1.0 那样“突变追随”。

---

## 6. 推荐参数起始值（防过冲）

- 模式：`CHASSIS_SPEED_PLAN_S_CURVE`
- `accel_x/y = 30~45`
- `accel_w = 15~25`
- `jerk_x/y/w = 0`（先用自动）
- 防侧翻：`enable=1, threshold=25~35, base=25~35, max=35~45`

按此起点做实车阶跃测试，再逐步提高响应速度。
