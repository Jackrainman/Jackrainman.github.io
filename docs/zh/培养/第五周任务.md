# 第五周任务

1. 学习CAN通信协议，了解使用STM32F429的CAN通信模块。F4相关资料在战队仓库中有。

2. 学习PID算法，了解PID算法如何对电机进行预期角度和位置控制。

3. 学习了解电机有关知识，搞清楚不同电机类型如伺服，步进，航模电机等，以及编码器，电调的作用，整理成笔记。

4. 查阅大疆数据手册，编写程序控制大疆3508/2006电机实现电机的速度环和位置环的控制（固定速度转动，转动到固定角度），官网上有相关demo自行参考。

以上任务有任何不理解的地方请及时向学长询问，关于电机资料等信息请自行到大疆官网查询数据手册，这一周是最后一次视觉和电控一起进行培训，请认真完成任务。
