## 一、培养路线图

```mermaid
flowchart LR
    A[新人入队] --> B[预备阶段]
    B --> C[第一阶段 基础能力]

    D --> E[第三阶段 项目能力]
    E --> F[参与真实机器人项目开发]
```

能力主线：

1. 预备阶段：统一环境、工具链、代码规范、最小外设基础
2. 第一阶段：做扎实基础控制链与程序结构
3. 第二阶段：完成模板迁移、模块化设计和综合整合
4. 第三阶段：完成 CAN、电机、PID、底盘与联调接入

## 二、推进原则

- 任务仍可按周发布，但培养主线按阶段推进
- 前两周默认归入预备阶段，不再作为完整培养主线的前两周
- 晋级依据是当前阶段是否通关，不是自然周数是否结束
- 每个阶段统一采用：`通关线 / 提升线 / 预研线`
- 快的同学可进入预研线，慢的同学走补救路径，不做公开排名
