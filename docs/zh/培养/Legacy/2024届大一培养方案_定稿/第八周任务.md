从本周开始电控和视觉的任务再一起布置，对电控有兴趣的可自行学习

1. 配置VScode的[edie插件]([简介 | Embedded IDE For VSCode](https://em-ide.com/docs/intro/))（恭喜你们终于要迈向现代ide了），注册github账号获取[DDL的stm32工程模板]([Deadline039/STM32-Template: STM32 project template.](https://github.com/Deadline039/STM32-Template))并上手使用（**这个部分视觉的同学也要做**）

2. 学习adc,dac模块知识，掌握相关例程代码的编写。

3. 掌握[蓝牙模块](https://blog.csdn.net/qq_46095529/article/details/119080699)，[nrf模块](https://www.ebyte.com/product/466.html)如何和配对与使用

4. 编写程序控制遥控器，要求能够在oled屏幕上显示两个摇杆x轴和y轴的读取值，按下按键能够显示对应按键的编号，同时能将数据通过蓝牙或者nrf成功发送出去。
