## 大一电控暑假学习任务：

必学部分：

1. 学习完毕freertos要求正点原子的全部例程都跟着做一遍（例程源码在实验室服务器上有自取）
2. 精进学习C语言（可以参考[Frank的课程](https://www.bilibili.com/video/BV1oJ4m1Y7ry?spm_id_from=333.788.recommend_more_video.0&vd_source=6bb92a819dfdcd7bd3bc4e88195b918d)），严格要求[代码规范](https://xju-hurricane-team.github.io/Electrical/C语言/Program-Design/C-code-style/)，学习了解C语言的内存模型，函数指针，泛性编程，和C语言重要库函数的使用方法(如 (f)(s)(v)scanf, (f)(s)(v)printf,atoi,strcat,strtol )  ，学到能够看懂canlist.c就是学懂了。
3. 复习这学期的学习任务，**遵循代码规范**（反复强调！！！），用vscode将这学期的学习任务全部进行复现。
4. 学习战队GitHub上的祖传代码（各类电机的驱动代码，通信协议，底盘的解算）
5. 嵌入式调试的原理，调试器，烧录器的进阶使用（stlink，[jlink]([UM08001 J-Link / J-Trace 用户指南 - SEGGER 知識庫](https://kb.segger.com/UM08001_J-Link_/_J-Trace_User_Guide#J-Link_Software_and_Documentation_Pack))，openocd的区别和联系），vscode中的launch.json如何配置。
6. stm32的启动流程，地址映射，时钟系统，中断的处理机制深入了解。

建议学习部分：

1. 深入了解stm32 hal库开发的框架构成，以及使用的各类编译器和调试器，有兴趣可以自己再clion搭建stm32开发环境会让自己对工具链更加熟悉。
2. 可以学习一门面向对象语言（python或者c++），理解面向对象和**层次化编程**
3. 可以学习一下Linux的基本使用方法（通过虚拟机，WSL，树莓派等方式都可）
