## 1. PID 控制原理

### 1.1 PID 概述

比例-积分-微分控制（PID）是工业控制中最广泛使用的控制算法。

### 1.2 三个环节

| 环节 | 作用 | 优点 | 缺点 |
|------|------|------|------|
| P (比例) | 误差成比例放大 | 响应快 | 存在静差 |
| I (积分) | 误差累积消除 | 消除静差 | 可能超调 |
| D (微分) | 误差变化率抑制 | 阻尼振荡 | 放大噪声 |

### 1.3 数学表达

$$u(t) = K_P e(t) + K_I \int_0^t e(\tau)d\tau + K_D \frac{de(t)}{dt}$$

### 1.4 PID 的局限性

- 参数整定依赖经验
- 对非线性系统效果不佳
- 难以处理模型不确定性

## 2. 状态空间基础

### 2.1 基本概念

**状态变量**: 完全描述系统所需的最少变量集合

**状态方程**:
$$\dot{x} = Ax + Bu$$

**输出方程**:
$$y = Cx + Du$$

### 2.2 状态空间 vs 传递函数

| 特性 | 传递函数 | 状态空间 |
|------|----------|----------|
| 适用对象 | 线性定常 | 线性/非线性 |
| 维数 | 单输入单输出 | 多输入多输出 |
| 初始条件 | 零初始 | 可考虑非零 |
| 物理意义 | 不够直观 | 清晰明确 |

### 2.3 能控性与能观性

- **能控性**: 输入能否控制所有状态
- **能观性**: 输出能否观测所有状态

## 3. 稳定性分析

### 3.1 BIBO 稳定性

**定义**: 有界输入产生有界输出

**判据**: 传递函数所有极点位于 s 平面左半平面

### 3.2 李雅普诺夫稳定性

**李雅普诺夫函数**: 正定标量函数 $V(x)$

**判据**:
- $\dot{V} < 0$ → 渐近稳定
- $\dot{V} \leq 0$ → 稳定

### 3.3 稳定裕度

- **相位裕度**: 增益为 1 时的相位与 -180° 的差值
- **增益裕度**: 相位为 -180° 时的增益与 1 的差值

## 4. 频域分析基础

### 4.1 波特图

描述系统频率响应，包含：
- **幅频特性**: $20\log|G(j\omega)|$
- **相频特性**: $\angle G(j\omega)$

### 4.2 奈奎斯特判据

根据开环 Nyquist 曲线判断闭环稳定性

### 4.3 带宽概念

**带宽频率**: 幅频特性下降到 -3dB 处的频率

**意义**: 系统响应速度的上限
