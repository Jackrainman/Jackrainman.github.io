## 🎯 学习路径推荐

### 路径一：标准学习路线（推荐）

适合大多数读者，循序渐进掌握控制理论：

```
PID 控制算法原理与应用
        ↓
控制理论基础
        ↓
何谓自抗扰 ──→ 何谓自抗扰 · 知识点前瞻（辅助）
        ↓
线性自抗扰控制（LADRC）详解
        ↓
LADRC 代码讲解
        ↓
高级控制理论
        ↓
LADRC 战队实战指南
```

### 路径二：快速上手路线

适合有一定基础，急需上手的读者：

```
PID 控制算法原理与应用
        ↓
何谓自抗扰 · 知识点前瞻
        ↓
线性自抗扰控制（LADRC）详解（重点阅读第 3、6 节）
        ↓
LADRC 代码讲解
```

### 路径三：深度研究路线

适合想要深入理解理论原理的读者：

```
控制理论基础（完整阅读）
        ↓
何谓自抗扰（完整阅读，结合视频）
        ↓
何谓自抗扰 · 知识点前瞻（复习总结）
        ↓
线性自抗扰控制（LADRC）详解（完整阅读，含数学推导）
        ↓
高级控制理论（全部章节）
```

## 💬 学习建议

1. **循序渐进**：不要跳过基础篇直接看 LADRC，状态空间知识很重要
2. **动手实践**：每学完一个算法，尝试在仿真或实物上实现
3. **数学推导**：不要怕公式，手动推导一次比看十遍理解更深
4. **多交流**：遇到问题及时与队友讨论，控制理论需要思维碰撞
