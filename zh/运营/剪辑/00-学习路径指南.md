## 📖 核心技能详解

### 1. 基础剪辑

| 项目 | 内容 |
|------|------|
| **难度** | ⭐⭐ |
| **预计时间** | 1-2 周 |
| **核心内容** | 界面熟悉、基础操作、快捷键 |
| **学习目标** | 能独立完成简单视频剪辑 |

### 2. 节奏与叙事

| 项目 | 内容 |
|------|------|
| **难度** | ⭐⭐⭐ |
| **预计时间** | 2-4 周 |
| **前置知识** | 基础剪辑 |
| **核心内容** | 音乐卡点、情绪把控、故事线构建 |
| **学习目标** | 视频有节奏感，观感流畅 |

### 3. 调色与包装

| 项目 | 内容 |
|------|------|
| **难度** | ⭐⭐⭐⭐ |
| **预计时间** | 3-4 周 |
| **前置知识** | Premiere Pro / DaVinci Resolve |
| **核心内容** | 一级调色、二级调色、字幕包装 |
| **学习目标** | 视频有质感，风格统一 |

### 4. 特效与合成

| 项目 | 内容 |
|------|------|
| **难度** | ⭐⭐⭐⭐⭐ |
| **预计时间** | 4-6 周 |
| **前置知识** | After Effects |
| **核心内容** | 绿幕抠像、MG 动画、特效合成 |
| **学习目标** | 能制作炫酷片头/转场 |

## 🚀 实战项目建议

### 新手阶段
- 混剪视频（3-5 个素材）
- 照片轮播 + 背景音乐
- 简单 vlog 剪辑

### 进阶阶段
- 产品宣传片（2-3 分钟）
- 活动纪录短片
- 品牌故事视频

### 高级阶段
- 商业广告片
- 品牌 TVC
- 创意短片

---

## 💡 学习建议

1. **多看多练**：每天刷热门视频，分析剪辑手法
2. **建立素材库**：分类整理常用素材，提升效率
3. **模仿创新**：从模仿优秀作品开始，逐步形成个人风格
4. **接受反馈**：多让他人观看，收集改进意见
5. **保持输出**：定期发布作品，建立个人作品集
